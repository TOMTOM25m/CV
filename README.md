# CV Portfolio

Dieses Repository enthält automatisch generierte Lebensläufe und wird durch den CV-Agenten gepflegt.

- **[Zur interaktiven Übersicht (GitHub Pages)](https://tomtom25m.github.io/CV/)**

## Struktur
- `index.html`: Übersichtsseite (Hub)
- `html/`: Enthält die HTML-Versionen der Lebensläufe
- `pdf/`: Enthält die PDF-Versionen der Lebensläufe
